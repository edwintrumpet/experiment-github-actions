# CD using GitHub Actions

Experimental project to learn continuous deployment using GitHub Actions, GitHub Pages and AWS S3

## Develop

Installation

```shell
npm i
```

Run in develop mode

```shell
npm start
```

## Deploy

Build static files

```shell
npm run build
```
